# Background Changer with React

A modern and responsive React project that automatically changes background images and animated text every 3 seconds to create an engaging landing page experience. Built using React, Tailwind CSS, and smooth state-based transitions.

---

## 🚀 Features

- 🔄 Automatic background image slider
- ✨ Dynamic text transitions
- 📱 Fully responsive design
- 🎨 Separate portrait backgrounds for mobile devices
- ⚡ Smooth UI experience using React Hooks
- 🖼️ Fullscreen immersive hero section
- 🎯 Navigation dots for manual slide switching

---

## 🛠️ Tech Stack

- ⚛️ React.js
- 🎨 Tailwind CSS
- 🧠 React Hooks (`useState`, `useEffect`)
- ⚡ Vite

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/SanjayLadumor/Background-Changer-with-React.git
```

Navigate to the project directory:

```bash
cd Background-Changer-with-React
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## 🧠 How It Works

- Background images and text are controlled using React state.
- `setInterval()` changes the current slide every 3 seconds.
- Different images are rendered for desktop and mobile devices.
- Tailwind CSS utilities handle responsiveness and fullscreen layouts.

---

## 📱 Responsive Design

This project includes:

- Desktop landscape backgrounds
- Mobile portrait backgrounds
- Overflow handling to prevent unwanted scrolling
- Adaptive layouts using Tailwind breakpoints
